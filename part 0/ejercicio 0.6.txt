Cuando un usuario abre una aplicación creada con React, el navegador primero solicita al servidor el archivo HTML inicial.
Este HTML contiene un contenedor vacío, normalmente un elemento <div id="root">.

Junto con el HTML, el navegador descarga los archivos JavaScript y CSS.
El JavaScript incluye tanto el código de la aplicación como la librería React, que se ejecutan para montar los componentes dentro del elemento root.

React utiliza un Virtual DOM para decidir qué partes de la interfaz necesitan actualizarse. Esto permite que los cambios se apliquen de forma rápida y eficiente sin recargar toda la página.
Gracias a este enfoque, la navegación y las interacciones dentro de la aplicación son fluidas y se gestionan principalmente en el lado del cliente.
